# Assets Directory

## Brand Colors

Based on the color logo analysis, the website uses these brand colors:

### Primary Colors (Teal/Cyan)

- Primary 500: `#009292` - Main brand teal
- Primary 700: `#006166` - Dark teal for depth

### Secondary Colors (Red)

- Secondary 500: `#FF0000` - Brand red for accents
- Secondary 900: `#350A0A` - Dark red/maroon

### Neutral Colors

- Neutral 950: `#000000` - Pure black from logo
- Various gray shades for text and backgrounds

### Usage Guidelines

- **Primary (Teal)**: Headers, buttons, links, brand elements
- **Secondary (Red)**: Call-to-action buttons, highlights, alerts
- **Neutral (Black/Gray)**: Text, backgrounds, subtle elements

## Images

Place your images in this directory:

- `/images/color-logo.png` - Color logo (main brand logo used in header and favicon)
- `/images/black-logo.png` - Black logo (alternative for light backgrounds if needed)
- `/images/white-logo.png` - White logo (for dark backgrounds like footer)
- `/images/hero-bg.jpg` - Hero background image
- `/images/about-team.jpg` - About section team photo
- `/images/services/` - Service-related images
- `/images/portfolio/` - Portfolio thumbnails
- `/images/team/` - Team member photos
- `/images/clients/` - Client logos

## Videos

Place your videos in this directory:

- `/videos/hero-video.mp4` - Main hero background video
- `/videos/hero-video.webm` - WebM version for better compatibility
- `/videos/demo-video.mp4` - Service demonstration video

## Recommended Image Sizes

- Hero background: 1920x1080px
- Service thumbnails: 500x300px
- Portfolio thumbnails: 600x400px
- Team photos: 400x400px (square)
- Client logos: 200x100px (maintain aspect ratio)

## Video Specifications

- Hero video: 1920x1080px, MP4/WebM, under 10MB
- Demo videos: 1280x720px, optimized for web

## Optimization Tips

1. Use WebP format for images when possible
2. Compress images to reduce file sizes
3. Use appropriate video codecs (H.264 for MP4, VP9 for WebM)
4. Consider using a CDN for large assets
